# 🚀 Credit Risk Prediction Web Application

This project is a Machine Learning-based web application that predicts whether a person is a credit risk based on input features.  
It uses a trained model and provides real-time predictions through a web interface.

---

## 📌 Project Overview

The goal of this project is to build a complete end-to-end Machine Learning pipeline:
- Data preprocessing
- Model training & evaluation
- Model selection
- Deployment using Flask

The application allows users to input details and get instant predictions.

---

## 🧠 Machine Learning Models Used

The following models were trained and evaluated:

- K-Nearest Neighbors (KNN)
- Random Forest
- Decision Tree
- Logistic Regression
- Support Vector Machine (SVM)
- AdaBoost
- XGBoost
- CatBoost

After comparison, the best performing model was selected based on accuracy.

---

## 🏆 Final Model Used

- **Model:** Random Forest Classifier  
- **Accuracy:** ~93.75%  
- The model was saved using `pickle` as `model.pkl`
